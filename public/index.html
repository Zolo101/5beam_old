API endpoints

GET/api/all---- Gets all the levels.

GET/api/level/:id---- Gets a specfic level.

GET/api/level/get/:id---- Gets the leveldata from the level specified.

/api/alland/api/level/:iddo not contain the actual leveldata, only the level info. For that, use/api/level/get/:id
