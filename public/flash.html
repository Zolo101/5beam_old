API endpoints

GET/api/flash/all---- Gets all the levels.

GET/api/flash/level/:id---- Gets a specfic level.

GET/api/flash/level/get/:id---- Gets the leveldata from the level specified.

/api/flash/alland/api/flash/level/:iddo not contain the actual leveldata, only the level info. For that, use/api/level/get/:id



/api/flash/*has a ratelimit of 50 requests per minute.

/api/flash/allhas a ratelimit of 20 requests per minute.



POST/api/flash/level/upload---- Uploads a level. Must have the correct format or it will not be accepted.

Structure:

{
name: string
author: string
data: any (the levelpack itself)
version: number
}

will be turned into this in the database:

{
id // Database ID
name // Name of levelpack
author // "Guest"
data // Levelpack
version // Structure level
}
